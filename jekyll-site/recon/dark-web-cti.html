Dark Web CTI Sources

Cyber Threat Intelligence (CTI) from the dark and deep web enables defenders to monitor threat actors, track ransomware groups, and identify leaked credentials before attackers weaponize them. For red teams, these sources also reveal TTPs, active exploits, and initial access brokers selling footholds into target organizations. This page catalogs the major source categories from the deepdarkCTI project.

CTI Source Categories

Dark/deep web CTI intelligence comes from three tiers:

  • Strategic — high-level intelligence about threat actor groups, campaigns, and industry targeting
  • Tactical — TTPs (MITRE ATT&CK), tools used, malware families, exploitation techniques
  • Operational — specific IOCs, leaked credentials, active infrastructure, breach announcements

Ransomware groups operate leak sites on the Tor network where they publish victim data. These sites are primary intelligence sources for active breach notifications and threat actor TTPs.

Parsing Ransomwatch Feed

# Check if a specific organization appears in ransomware victim lists:
proxychains4 curl -s https://api.ransomwatch.telemetry.ltd/v2/posts | \
  python3 -m json.tool | grep -i "TARGET_COMPANY_NAME"

# Pull recent victims (last 30 days):
proxychains4 curl -s https://api.ransomwatch.telemetry.ltd/v2/recentvictims | \
  python3 -c "
import sys, json
data = json.load(sys.stdin)
for v in data[:20]:
    print(f\"{v.get('post_title','?')} — {v.get('group_name','?')} — {v.get('published','?')}\")"

CTI Methodology: Threat Research Workflow

#!/bin/bash
# Dark web CTI sweep for a target organization

TARGET="targetcompany.com"
COMPANY="Target Company"

echo "[1] Check ransomware victim lists..."
curl -s https://api.ransomwatch.telemetry.ltd/v2/posts 2>/dev/null | \
  python3 -m json.tool 2>/dev/null | \
  grep -i "$COMPANY" && echo "WARNING: Found in ransomware database"

echo "[2] Search breach databases..."
# IntelX: https://intelx.io/?s=$TARGET
# DeHashed: https://dehashed.com/search?query=@$TARGET
echo "Manual check: https://intelx.io/?s=$TARGET"
echo "Manual check: https://dehashed.com/search?query=@$TARGET"

echo "[3] Check stealer log services..."
echo "Manual check: https://whiteintel.io (requires account)"
echo "Manual check: https://cavalier.hudsonrock.com/ (requires account)"

echo "[4] Search exploit databases for target technology..."
# Requires knowing target tech stack (from recon)
searchsploit "TARGET_TECHNOLOGY VERSION"

echo "[5] Monitor Ahmia for onion mentions..."
proxychains4 curl -s "https://ahmia.fi/search/?q=$COMPANY" 2>/dev/null | \
  grep -oE 'http[^"]+\.onion[^"]*' | sort -u
